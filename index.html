Viewing the HTML source of a webpage in Chrome

  1. Open Google Chrome
  2. Navigate to the web page you want to investigate
  3. If using a Mac, right-click on the page
  4. Select "View Page", and that's it for Mac users!
  5. If using Windows, press CTRL+U
  6. Select "Tools"
  7. Select "View Source" and you're done!
